Allmoxy Top Edge Report

Shop-floor reporting, order control, and cut optimization for DBS Drawers.

What Was Built

  • Offline HTML calculator for Allmoxy CSV exports.
  • Drag and drop CSV import.
  • Grouped print reports by department.
  • Order remove and restore controls.
  • CSV export and print-ready output.
  • Compact cut optimization for non-solid materials.
  • Sync Report to Saw button for dashboard delivery.

Why It Helps

  • Less manual sorting by the operator.
  • Clearer department labels.
  • More accurate parts-to-boxes totals.
  • Better visibility into sheet quantity and waste.

Order Management

  • Active orders can be removed with multi-select dropdowns.
  • Removed orders can be restored later.
  • The report recalculates using only active orders.

Report Labels

Visible categories were simplified:

Plywood Solid FAA MDF / PBC

PVC and wood tape remain top-edge values, not category labels.

Calculation Fixes

  • Inch marks in material names now parse correctly.
  • Maple materials are no longer skipped.
  • Parts are summed before converting to boxes.
  • Box totals no longer inflate from row-by-row rounding.

Cut Optimization Rules

  • Rips run in the sheet length direction.
  • Rip heights are packed across sheet width.
  • Trim is 0.25" on both sides of width.
  • Kerf is 0.188".
  • Solid and FAA material are excluded.

Saw Sync

  • The calculator sends reports to http://server24:8787/sync-report.
  • The helper saves reports into the shared server22 saw folder.
  • The saw operator opens one dashboard link from the shared folder.
  • This removes manual floor paperwork once the helper/dashboard are in place.
